# ESP32 Media Viewer

ESP32 Media Viewer lets you convert images, GIFs, and videos on Windows, then flash or upload them to supported Waveshare ESP32 display boards.

## Downloads

Use the latest release:

- `ESP32MediaViewerCompanion-Full-Setup.exe` is recommended. It includes the app, FFmpeg, esptool, and firmware files.
- `ESP32MediaViewerCompanion-Web-Setup.exe` is smaller and downloads the selected tools and firmware during setup.

## Supported Boards

- Waveshare ESP32-S3 1.3inch Display Development Board

## Install Notes

Run the setup EXE, choose an install folder, and accept the User Agreement. The app stores settings beside the installed app when possible, writes generated media to an `output` folder by default, and includes an uninstall entry in Windows Apps & features.
